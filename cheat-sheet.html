Create Database

  • Show table

    SELECT * FROM table;

  • Create table

    CREATE TABLE table_name

    • Example CREATE TABLE Persons (
    P_Id int NOT NULL,
    LastName varchar(255) NOT NULL,
    FirstName varchar(255),
    Address varchar(255),
    City varchar(255),
    PRIMARY KEY (P_Id) )
  • Insert/add to table

    INSERT INTO table_name

    VALUES(value#1, value#1…);

  • Delete

    DELETE FROM table_name

    WHERE some_column = some_value;

  • Create index

    CREATE INDEX index_name;

Query Database

  • Select from table
    • Between

      SELECT value FROM WHERE coulomb_name BETWEEN value_1 AND value_two;

    • Like

      SELECT value FROM table WHERE coulomb_name LIKE pattern;

    • Wildcards

      SELECT * FROM table WHERE value LIKE “wildcard”

      • %- A substitute for zero or more characters
      • _ A substitute for a single character
    • AND/OR

      SELECT column_name(s) FROM table_name WHERE condition AND|OR conditionOrder by/group by

    • ORDER BY

      coulomb_name ASC/DESC

  • Joins
    • Inner Join/ Join

      Returns all rows when there is at least one match in BOTH tables

    • LEFT JOIN

      Return all rows from the left table, and the matched rows from the right table

    • RIGHT JOIN

      Return all rows from the right table, and the matched rows from the left table

    • FULL JOIN

      Return all rows when there is a match in ONE of the tables

    SELECT Orders.OrderID, Customers.CustomerName, Orders.OrderDate
    FROM Orders INNER JOIN Customers ON Orders.CustomerID=Customers.CustomerID;
